# Basic API

A Basic API made with PHP without any Framework

## Uses

- PHP 8.0
- PHP League - Container
- PHP League - Route
- Laminas Diactoros
- PHPUnit

## How to Run

To execute this project can be made two ways.

Without `docker-compose`:

```bash
composer install
php -S localhost:8080
```

With `docker-compose`:

```bash
docker-compose up -d
```
